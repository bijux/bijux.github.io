# Bijux Core

`bijux-core` is the execution and governance backbone for the public
Bijux system family.

## Open Core When

- the question is about CLI behavior, DAG execution, or repository-wide runtime discipline
- the answer likely lives in contracts, evidence, or release governance
- you need the shared backbone rather than a domain-specific product layer
