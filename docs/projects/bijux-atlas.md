# Bijux Atlas

`bijux-atlas` is the delivery and operational surface for services,
datasets, APIs, and docs control-plane behavior.

## Open Atlas When

- the question is about API delivery, dataset publishing, or service behavior
- you are tracing docs UX, ops validation, or operational evidence
- you need the repository that owns the docs platform control plane
