# Bijux Canon

`bijux-canon` is the governed knowledge-system stack for deterministic
ingest, retrieval, reasoning, orchestration, and controlled runtime
acceptance.

- the question is about ingest, indexing, reasoning, agents, or runtime proof
- the issue is in a governed AI workflow rather than a domain product shell
- you need repository or package handbooks for Canon’s split architecture
