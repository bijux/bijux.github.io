    What Each Repository Covers¶

    Repository Open it for
    Bijux Core execution backbone, release evidence, and governance that do not disappear into shell glue
    Bijux Canon contract-first knowledge-system design with visible package boundaries and compatibility discipline
    Bijux Atlas data-service architecture, immutable delivery thinking, and operationally serious public surfaces
    Bijux Proteomics scientific product work that keeps structure under laboratory and discovery constraints
    Bijux Pollenomics evidence-heavy domain modeling in an unusual archaeology-facing context
    Bijux Masterclass the ability to teach architecture, workflow rigor, and programming design without flattening them
