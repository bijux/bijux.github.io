# Stewardship

The hub is part of the documentation product, not a separate landing
page experiment. These pages explain the quality rules that keep the hub
aligned with the repository handbooks it points to.

## Stewardship Focus

- keep the hub navigation contract aligned with the project sites
- keep routes and labels stable enough for long-term reuse
- keep the hub honest by routing readers quickly instead of duplicating repository proof
